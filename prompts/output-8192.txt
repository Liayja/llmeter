请用 Python 从零实现一个完整的分布式键值存储系统（Distributed KV Store）。要求包含以下全部模块，每个模块都需要完整的代码实现和详细注释：

## 1. 网络通信层
- 基于 asyncio 的 TCP 服务器和客户端
- 自定义二进制协议：支持 SET/GET/DELETE/EXISTS/KEYS 五种命令
- 协议格式：4 字节长度前缀 + JSON 载荷
- 连接池管理

## 2. 存储引擎
- 基于 LSM-Tree 的存储引擎
- MemTable：使用红黑树（自己实现，不用现成的 sortedcontainers）
- SSTable：磁盘上的不可变有序文件，包含索引块和数据块
- Compaction 策略：Size-Tiered Compaction
- WAL（Write-Ahead Log）：保证崩溃恢复

## 3. 一致性哈希环
- 虚拟节点机制（每个物理节点 150 个虚拟节点）
- 节点加入/离开时的数据迁移逻辑
- 使用 MD5 哈希函数

## 4. 数据复制
- 主从复制模型，写入主节点后异步复制到 N 个副本
- Hinted Handoff：节点暂时不可用时的临时接管机制
- Read Repair：读取时检测并修复不一致数据

## 5. 故障检测与恢复
- 基于 Gossip 协议的成员管理
- 心跳机制（每 1 秒）
- 故障检测：Phi Accrual Failure Detector

## 6. 客户端 SDK
- 连接路由（根据 key 找到负责的节点）
- 重试策略：指数退避，最多 3 次
- 请求超时处理

## 7. 测试
- 单元测试：每个模块至少 3 个测试用例
- 集成测试：3 节点集群的完整读写、故障切换测试

请给出所有模块的完整 Python 代码，不要使用 TODO 或伪代码占位。
